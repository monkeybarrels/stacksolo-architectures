RAG Platform Chat Widget

This page demonstrates the embeddable chat widget. Look for the chat button in the bottom-right corner of the screen.

Integration

To add the widget to your website, include this code before the closing </body> tag:

<script>
  window.RAGWidgetConfig = {
    botId: 'YOUR_BOT_ID',
    apiUrl: 'https://your-api-domain.com',

    // Optional customization
    position: 'bottom-right',    // or 'bottom-left'
    theme: 'light',              // or 'dark'
    title: 'Chat with us',
    primaryColor: '#4361ee',
    welcomeMessage: 'Hi! How can I help you today?',
  };
</script>
<script src="https://your-domain.com/widget.js"></script>

Configuration Options

Option Type Required Description
botId string Yes Your bot's unique identifier
apiUrl string Yes Base URL of your RAG Platform API
position string No 'bottom-right' (default) or 'bottom-left'
theme string No 'light' (default) or 'dark'
title string No Header title (default: 'Chat')
primaryColor string No Hex color for button/header (default: '#4361ee')
welcomeMessage string No First message shown when widget opens
